# LTS Network Analysis Bicycle Network Zurich

Summarized Findings: https://storymaps.arcgis.com/stories/2a04009ba3c94820aed8bdc6fab6f2bb

## Code:
This repository contains code and tools for analyzing Level of Traffic Stress (LTS) in the bicycle network of Zurich. The main analysis is performed in the Jupyter notebook `LTS_network_analysis.ipynb`, with supporting functions in `thesis_functions.py`.

Please be aware that code quality may be (definitely is) suboptimal.
